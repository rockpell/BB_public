# BB_public

총알을 이용하여 맵에 존재하는 목표물을 모두 파괴하는 퍼즐게임입니다.

총알을 발사한 이후에 다시 터치하면 총알이 멈추고 총알을 다시 발사할 수 있는 요소가 있습니다.

이미지와 사운드의 라이센스 문제로 인해 스크립트만을 별도로 올린 저장소입니다.

## 개발환경
Windows10(64bit)

## 작성언어
C#(Unity)

## 사용 IDE
Unity 2018.1.6f

Visual Studio 2017
